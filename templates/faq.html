Frequently Answered Questions

Why is this beta test so formal?
Apple's ad-hoc distribution (distribution that does not go through the App Store) to be distributed to 100 devices in total.

Is this a closed beta test?
No. This beta test is quite open. However, only devices that are on the list of participating devices will be able to use the binaries provided on this page.

I'm an iPhone developer, and can run the binaries provided on this page without being added to your ad-hoc distribution profile. Should I sign up for the beta test anyway?
Please don't. This beta test is only for people who cannot build the source and run it on their phones themselves. We'd still love any feedback we can get, including bug reports. See the Bug Reporting page.

What is required of a participating beta tester?
We expect you to upload any crashes that happen to you. Crash reports on iOS are synced to the computer on which you use iTunes to manage your device. We provide a downloaable utility that helps you upload crash reports to the beta test portal. See Reporting Crashes.

Apart from that hope you will report your thoughts on the program and any bugs you find and any features you would like to see in the program to the GitHub issue trackers. See the Bug Reporting page to find out how to submit feedback.

Why do I need to provide my device's UDID?
Devices are identified by their UDID when using Apple's ad-hoc distribution mechanism.
