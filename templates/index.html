Mumble for iPhoneOS

This is the home of the iPhoneOS port of Mumble. That is, a Mumble client that runs on the iPhone, iPod Touch and (maybe in the future), iPad. Mumble is an open-source, low-latency, high-quality voice chat software.

This site is primarily a portal for beta-testing efforts of the iPhoneOS client.

For Mumble-related information, please visit the Mumble website.

The Source

To check out the code from GitHub: 
   git clone http://github.com/mkrautz/mumble-iphoneos.git
The code for this website is also available on GitHub: 
   git clone http://github.com/mkrautz/mumble-iphoneos-betaweb.git
